Credit by: ***Net Ninja*** (link: https://www.youtube.com/playlist?list=PL4cUxeGkcC9iurLoO9Mu7GqsKlxEXcf8m)

This repository aims to reconstruct Tetris clone using Python as an introductory project to a high-school student
to get familiar with the concept of ***Object-Oriented Programming*** (OOP). one of the fundamental concept of
programming style as a preparation for their pursuit of Computer Science major career path.

## Highlight Features

#### 1. **The OOP code**
This game is built using classes and objects, helping students understand core OOP concepts such as encapsulation, inheritance, and modular code structure.
#### 2. **The File Integration in Python**
This repository demonstrates how to organize and manage external files (e.g., assets, configurations, or score tracking) within a Python project, reinforcing good coding practices
#### 3.**Dual Platform Experience**
This game is functioned on 2 platforms:
* **Visible platform:** A playable Tetris game window built with pygame, where students can interact with blocks, scoring, and gameplay.
* **Invisible Platform (or I call it the matrix):** The underlying OOP codebase that runs the game logic, offering students insight into how the mechanics are implemented behind the scenes.
## Conclusion
This project is a learning tool as much as it is a game. 
By exploring the code, students will:
* Understand how classes (and super classes) and objects work in Python
* Learn how to structure a program using OOP principles
* A new way to develop the games: functioning in 2 different platforms (visible and invisible)
* Gain confidence in building interactive applications (simple math is applied in this project)
